# Atividade: Estrutura e Seletores CSS

Este projeto apresenta os conceitos fundamentais de CSS solicitados na atividade acadêmica.

## 1. Conceito de CSS Interno e Externo
- **CSS Interno:** Estilos declarados dentro da tag `<style>` no `<head>` do HTML. Centraliza o código em um único arquivo.
- **CSS Externo:** Estilos declarados em arquivos externos `.css`, permitindo que várias páginas compartilhem o mesmo layout.

## 2. O que são Seletores CSS?
Seletores são padrões que conectam as declarações de estilo (cores, fontes, tamanhos) aos elementos específicos do HTML (títulos, parágrafos, divs).

## 3. Os 5 Tipos de Seletores Demonstrados:
1. **Universal (*):** Altera todos os elementos de uma vez.
2. **Elemento:** Altera tags específicas (ex: `h1`).
3. **ID (#):** Altera um único elemento exclusivo.
4. **Classe (.):** Altera um grupo de elementos semelhantes.
5. **Atributo ([]):** Altera elementos com base em suas propriedades (ex: links externos).

## 4. CSS Inline
Consiste em aplicar o estilo diretamente na tag HTML através do atributo `style`.
