# CSS-atividade
Conceito de CSS
CSS Interno: As regras de estilo são colocadas dentro da tag <style>, que fica no <head> do próprio arquivo HTML. É útil para páginas únicas.

CSS Externo: Os estilos ficam em um arquivo separado (ex: style.css) e são vinculados ao HTML via tag <link>. É a melhor prática para manter o código organizado e reutilizável.

O que são Seletores CSS?
Seletores são os padrões usados para "selecionar" os elementos HTML que você deseja estilizar. Eles dizem ao navegador: "aplique esta cor neste elemento específico".

5 Tipos de Seletores
Seletor de Elemento (ou Tipo): Alvos baseados no nome da tag (ex: p, h1, div).

Seletor de Classe: Alvos marcados com o atributo class (ex: .botao-azul). Usa-se o ponto . antes do nome.

Seletor de ID: Alvo único marcado com id (ex: #cabecalho-principal). Usa-se a cerquilha #.

Seletor Universal: Seleciona todos os elementos da página. Representado pelo asterisco *.

Seletor de Atributo: Seleciona elementos com base em um atributo ou valor de atributo (ex: input[type="text"]).
