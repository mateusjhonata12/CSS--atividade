Definições de CSS e Seletores

CSS Inline: Este bloco usa estilo direto na tag. É útil para excessões, mas evita-se em projetos grandes.
CSS Externo: (Conceito) Os estilos ficam em um arquivo .css separado e são vinculados via tag <link>. É o padrão profissional.
O que são Seletores? São as regras que indicam ao navegador quais elementos HTML devem receber a estilização definida no CSS.
  • 1. Universal (*): Altera todos os elementos de uma vez.
  • 2. Elemento (section): Define o corpo branco do conteúdo.
  • 3. ID (#lista-seletores): Remove as bolinhas desta lista.
  • 4. Classe (.card-conceito): Estiliza estes quadros azuis repetidos.
  • 5. Atributo (a[target]): Estiliza links que abrem em nova aba.
